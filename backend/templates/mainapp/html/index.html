Unet Segmentation Trainig

Description

My educational project, in which I trained different models on the COCO dataset to segment people into images, using different data augmentation and acceleration techniques using CUDA. As a pre-trained model, priority fell on the Unet-VGG16 architecture.

Мой учебный проект, в котором я тренировал разные модели на датасете COCO на сегментацию людей на изображений, применяя разные техники аугментации данных и ускорения с помощью CUDA. В качестве предобученной модели приоритет пал на Unet-VGG16 архитектуру.

Tools

Rpi4/Esp32 Computer Vision Integration

Description

In this project, communication was created between raspberry pi4/Esp32 mycropython boards and remote servers that receive images, perform object recognition, segmentation, provide the ability to manually control robots based on these boards, or use PID controllers to make decisions about movement.

В этом проекте была создана коммуникация между платами raspberry pi4/Esp32 mycropython и удаленными серверами, которые принимают изображение, занимаются распознаванием объектов, сегментацией, дают возомжность ручного управления роботами на базе этих плат или с помощью PID регуляторов сами принимают решения о движении.

Tools

  • Raspberry Pi 4, rpi.gpio

  • Esp32, Mycropython

  • Django, Channels/Daphne, Redis

  • YoloV8

  • transformers, AutoImageProcessor, Mask2FormerForUniversalSegmentation

  • Numpy, CuPy, OpenCV
